Javascript30

Demo site and learnings from the 30 Javascript tutorials in the Javascript30 course from Wes Bos. No compilers, no frameworks, no libraries - just vanilla Javascript. I’ve refactored and updated the CSS for consistent styling.

This demo site is held together by Hugo and the beautiful street photography is by Steve Roe.

By Alice.
